Rules for naming JS variables

Also check Javascript File for Legal and Illegal Variable Names

5 Legal Variable Names

  • let userName
  • let _age
  • let $price
  • let studentCount
  • let total5

5 illegal Variable Names

  • let 1name
  • let user-name
  • let var
  • let user name
  • let @price

Variable names can only contain dolor $ , underscore _ letters and Digits

For example $my_1stVariable


Variables must begin with a $ Dolor underscore _ or Digits

For example $name, _name or name


Variable names are case case sensitive


Variable names should not be JS Keywords
